# Hebrew Shorts Generator

AI platform to create and manage Hebrew TikToks, Reels, and Shorts. Paste a concept to auto-generate scripts, realistic Hebrew voiceovers, and customizable video assets tailored to your niche.

## Key Features
- **Native RTL Engine:** Perfectly fixes backward text and punctuation for Hebrew.
- **Auto-Scripting:** Generate viral-ready scripts from a simple concept.
- **Voiceover Integration:** Realistic Hebrew AI voices.
- **Channel Scaling:** Automate your content creation workflow.
